Getting Started

Software Requirements

You will need the following tools:

  • Git

    The labs are on GitHub, so you will need Git to get to them. You can download Git here.
    An excellent tutorial for git can be found here.

    To clone the scala-labs repo, use the following command:
    git clone git://github.com/scala-labs/scala-labs.git

  • Maven 2.2.x

    The Scala-labs projects are built using Maven 2. Get it here. The build has been verified with Maven 2.2.1. Older versions might also work but have not ben tested

  • A Java VM

    Scala compiles to JVM bytecode, so you will need a JVM to run any of the labs. Download it here.
    or use the one that comes built into your OS. We have tested on Java 5 and Java 6 without problems. Older versions are unlikely to work.

  • Scala

    Since this is a Scala lab, you will need a basic Scala installation. To get one, go to the scala download page and follow the directions for your platform.

  • IDE Support

    There are scala plugins available for Eclipse, IntelliJ and Netbeans, but be warned: the maturity of these plugins is not exactly on par with the Java support that you might be accustomed to.

    At our live Scala Labs sessions we use IntelliJ Community Edition (IdeaCE) and nightly builds of the Scala plugin since they are the most stable and feature complete at the moment.

    The latest versions that we have tested and which work together well are: IdeaCE 90.193 (the latest beta as of November 25th 2009) and nightly build number 190 of the Scala plugin. To install the plugin, simply unzip it and copy the resulting "Scala" directory to the plugins directory of Idea.

    Of course, you can always use your favorite text editor. Some of us even switched back to emacs.

The labs

After getting the code with Git (see above), you will have two maven projects called "labs" and "solutions". The "labs" project has a lot of failing unit tests and your job is to fix them, starting with the super-simple first test in the first basic lab, which features our old friend "hello world". The "solutions" project contains our solutions in case you get stuck.

There is an IntelliJ Idea project included in the "labs" and "soluions" projects. You might have to define a JDK in the project settings before the project will compile.

Problems ?

If you encounter any problems, please join us at our scala-labs google group. Maybe we can help sort them out.
