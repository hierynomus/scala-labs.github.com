Getting Started

OK, so you want to get some hands-on Scala coding done. Here's what you need to get started with the labs:

  • Git

    The labs are on GitHub, so you will need Git to get to them. You can download Git here.
    An excellent tutorial for git can be found here. But for now a basic svn-style checkout will do. If you copy the following command into your terminal you will accomplish just that:
    git clone git://github.com/scala-labs/scala-labs.git

  • Maven 2.x

    The Scala-labs projects are built using Maven 2. Get it here.

  • A Java VM

    Scala compiles to JVM bytecode, so you will need a JVM to run any of the labs. Download it here.

  • Scala

    Since this is a Scala lab, you will need a basic Scala installation. To get one, go to the scala download page and follow the directions for your platform.

  • IDE Support

    There are scala plugins available for Eclipse, IntelliJ and Netbeans, but be warned: the maturity of these plugins is not exactly on par with the Java support that you might be accustomed to. At our live scala labs sessions we use the IntelliJ plugin, because that one seems to be working the smoothest for now.
    Of course, you can always use your favourite text editor. Some of us even switched back to emacs.

Problems

If you encounter any problems, please join us at our scala-labs google group. Maybe we can help sort them out.
